## Getting Started

After cloned the project

Install dependencies

```bash
npm install
# or
yarn install

```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
